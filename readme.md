## Description

Created to observe a discord channel, list the users and channel join date

## Start project

- Install dependencies

  ```
  $ npm install
  ```

- Set .env file

  ```
  MODE=""
  DISCORD_TOKEN=""
  CLIENT_ID=""
  GUILD_ID=""P
  ```

- Add commands to bot

  ```
  $ npm run commands
  ```

- Run bot
  ```
  $ npm start
  ```
